# Video Dataset Scraping Suite

A collection of Python scrapers for building video datasets from various cinematography and video platforms. For research and educational purposes.

## Scrapers

| Platform | Status |
|----------|--------|
| [ShotDeck](shotdeck/) | Active |
| [Frameset](frameset/) | Planned |
| [Film.ai](filmai/) | Planned |
| [YouTube](youtube/) | Planned |
| [Vimeo](vimeo/) | Planned |

## Getting Started

### Prerequisites

- Python 3.10+
- pip

### Installation

```bash
pip install -r requirements.txt
```

### Usage

Navigate to the desired scraper folder and follow its README for specific instructions:

```bash
cd shotdeck/
python shotdeck_scraper_fast.py
```
